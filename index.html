Git Quiz

What is Git and why is it so important?

Git is a version control system that allows collaboration between developers.

What's a branch and why do we create them?

A branch is a divergence from the main line of development so you can continue to do work without messing up the main line. We creat them using the command $ git checkout -b BRANCHNAME

What's the working tree?

The working tree is the folder where the repo is initialised.

What's the staging area?

The staging area is the area that holds all the files we're ready to commit and push to the repository.

What's the command to retrieve the latest version of your files from the remote server?

$ git pull origin BRANCHNAME
